# Get-ChocoPackages

## SYNOPSIS
Get the list of all installed chocolatey packages.

## SYNTAX

```
Get-ChocoPackages [<CommonParameters>]
```

## DESCRIPTION
Get the list of all installed chocolatey packages.

## EXAMPLES

### EXAMPLE 1
```
Get-ChocoPackages
Name             Version
----             -------        
github-desktop   3.2.3
vlc              3.0.18
```
