# Get-ChocoSources

## SYNOPSIS
Get the list of chocolatey sources.

## SYNTAX

```
Get-ChocoSources [<CommonParameters>]
```

## DESCRIPTION
Get the list of chocolatey sources.

## EXAMPLES

### EXAMPLE 1
```
Get-ChocoSources

----                ---                                                 --------     ----------- ----------- ---------
```
